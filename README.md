# Mining con la WX 7100

Ciao!

I file li trovati sulla sezione RELEASE qui a destra --->

Ho inserito anche i bios che ho utilizzato:

- il secondo bios è il bios che ho moddato utilizzando polaris bios editor 1.7.6

Buon mining!
